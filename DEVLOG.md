# Devlog

This file tracks broader public thoughts and milestones during development of **Once Upon a Tale**.

It is not meant to be a spoiler document or a full production diary.

## Entry 01 — Project Direction

Once Upon a Tale is being developed as an original otome project with a strong focus on atmosphere, emotional route design, branching narrative structure, and replayability.

Current development is centered on building a solid foundation before committing too early to implementation details that may need to change later.

That means a lot of the current work happens in areas like:
- narrative structure
- systems planning
- character development
- worldbuilding
- presentation direction

The goal is not just to make a game with romance routes, but to build an experience where player choices feel personal and where the tone, mechanics, and story all support each other.
