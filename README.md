# Once Upon a Tale

**Once Upon a Tale** is an original otome game project currently in development.

Set in a fantasy world shaped by stories, memory, choice, and identity, the game combines branching narrative routes, character-driven romance, stat-based progression, and interactive systems designed to make each playthrough feel personal.

This project is being developed as a solo passion project, with work spanning writing, visual design, systems planning, and implementation.

## Current Status

**Stage:** Early development / pre-production  
**State of the project:** Active concept, writing, systems planning, and prototype work

The game is still evolving, which means story structure, art direction, mechanics, and interface elements may change over time.

## Genre

- Otome
- Narrative adventure
- Romance
- Fantasy
- Choice-driven
- Stat / route-based progression

## Core Pillars

- **Character-driven routes** with emotional depth and distinct relationship dynamics
- **Branching choices** that shape tone, trust, and narrative outcomes
- **Worldbuilding woven into the story**, rather than treated as background decoration
- **Progression systems** that support roleplay and route access
- **Mystery and emotional tension** beneath a deceptively inviting surface

## Planned Features

- Branching dialogue choices
- Multiple romance routes
- Route-specific events and endings
- Hidden conditions and unlockable content
- Stat-based progression
- Minigame integration
- Save / load functionality
- Original UI and presentation design
- Character artwork and CG/event illustrations
- Soundtrack and sound design integration

## Project Goals

The goal of **Once Upon a Tale** is to create an original otome experience that feels emotionally engaging, replayable, and narratively layered, while combining strong atmosphere with meaningful player agency.

The project aims to balance:
- romance and mystery
- comfort and unease
- charm and consequence
- familiar fairytale motifs with more unsettling or introspective undertones
