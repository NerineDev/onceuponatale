# Contributing

Thank you for your interest in **Once Upon a Tale**.

At this time, the project is being developed as a **solo project** and is **not open to external contributions** for writing, art, or code.

## Current Policy

Please do not submit:

- pull requests with story content
- rewritten dialogue or route ideas
- character redesigns
- art assets
- code contributions intended for direct integration
- public forks presented as official extensions of the project

## Why

Because this is a narrative-heavy original project, consistency matters a lot across:

- tone
- character writing
- worldbuilding
- route structure
- visual direction
- project scope

Keeping development centralized helps preserve that consistency during early development.

## Feedback

General feedback, encouragement, or respectful interest is appreciated.

If public issue tracking is enabled in the future, it may be used for:
- typo reports in public documentation
- broken public links
- repository housekeeping issues

## Important Note

Please do not repost, repurpose, or redistribute project materials as if they were free-to-use resources.

See [RIGHTS.md](RIGHTS.md) for more information.
